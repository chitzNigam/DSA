# 🕸️ Graph

## 🔍 What is it?
A **Graph** is a set of **nodes (vertices)** connected by **edges**. Can be:
- Directed / Undirected
- Weighted / Unweighted

## 🧠 Representations
- Adjacency List
- Adjacency Matrix

## 📚 Use Cases
- Social networks
- Route/path finding
- Web crawling

## 📝 Notes
- Traversals: DFS, BFS
- Algorithms: Dijkstra, Floyd-Warshall, Kruskal, Prim
