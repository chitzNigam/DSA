# 📐 K-D Tree

## 🔍 What is it?
A **K-Dimensional Tree** is used for organizing points in k-dimensional space.

## ✅ Key Operations
- `insert(point)`
- `rangeQuery(box)`
- `nearestNeighbor(point)`

## 📚 Use Cases
- 2D/3D spatial queries
- KNN classification

## 📝 Notes
- Static structure; not great for dynamic updates
- Time: O(log n) average
