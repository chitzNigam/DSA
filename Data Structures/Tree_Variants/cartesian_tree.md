# 🧭 Cartesian Tree

## 🔍 What is it?
A **Cartesian Tree** is a binary tree with:
- In-order traversal equal to original sequence
- Min-heap or max-heap properties

## ✅ Key Operations
- `buildFromArray()`
- `rangeMinimumQuery()`

## 📚 Use Cases
- RMQ preprocessing
- Parsing expressions

## 📝 Notes
- Can be built in linear time
- Used in RMQ optimizations
