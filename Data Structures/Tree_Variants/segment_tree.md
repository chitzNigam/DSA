# 📊 Segment Tree

## 🔍 What is it?
A **Segment Tree** allows fast range queries and point updates on arrays.

## ✅ Key Operations
- `build()`
- `rangeQuery(l, r)`
- `update(i, val)`

## 📚 Use Cases
- Range sum/min/max queries
- Frequency queries

## 📝 Notes
- Time: O(log n) for query and update
- Requires 4n space
