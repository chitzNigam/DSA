# 📏 Interval Tree

## 🔍 What is it?
An **Interval Tree** efficiently finds all intervals that overlap a query interval.

## ✅ Key Operations
- `insert(interval)`
- `delete(interval)`
- `search(overlap)`

## 📚 Use Cases
- Scheduling
- Computational geometry
- DNA alignment

## 📝 Notes
- Based on balanced BST
- Stores max-end in subtree
