# 📂 B-Tree / B+ Tree

## 🔍 What is it?
Multi-way balanced trees used for block-based data access. 
- **B-Tree**: Internal + leaf nodes contain keys
- **B+ Tree**: Only leaves contain values

## ✅ Key Operations
- `search(key)`
- `insert(key)`
- `delete(key)`

## 📚 Use Cases
- Databases
- File indexing
- Operating systems

## 📝 Notes
- Height is low even for large data
- Optimized for disk I/O
