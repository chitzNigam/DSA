# 🧠 Dancing Links (DLX)

## 🔍 What is it?
A technique to efficiently implement backtracking for exact cover problems using a doubly linked list.

## ✅ Key Operations
- Remove / Restore nodes
- Efficient backtracking

## 📚 Use Cases
- Sudoku solving
- Polyomino tiling
- Exact cover problems

## 📝 Notes
- Introduced by Donald Knuth
